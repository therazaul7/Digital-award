# HTML Development KIT

## Requirement  
- Nodejs [HERE](https://nodejs.org/en/download/)

## install
- Clone git repository
```shell script
git clone git@github.com:therakib7/html-dev-kit.git
cd html-dev-kit
```
- Install Node package
```shell script
npm install
```

## Changes
- You need to change html-dev-kit name 

## NPM Helper comment
```shell script
npm run dev
```
```shell script
npm run prod
```  
```shell script
npm run watch 
``` 
```shell script
npm run watch-pro
``` 
